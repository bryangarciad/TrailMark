The architecture that kills the redundancy

One shared core, two platforms

Course 1 builds the iPhone app and a shared local Swift package, TrailmarkCore — models, the HealthKit manager, the route engine, the media store, motion, and connectivity. Course 2 adds a watchOS target that simply imports it, so the same logic is never rewritten for two platforms. Course 3 takes on the genuinely watch-hard topics: live workout sessions, wrist-to-pocket sync, power tuning, and complications.

CoreLocation / MapKit Core Motion HealthKit (read + write) AVFoundation media WatchConnectivity Swift Charts WidgetKit complications Shared Swift package

Trailmark is a working name — rename it to make it your own. Built for Xcode 26 · iOS / watchOS 26. See the Setup Guide to begin.
